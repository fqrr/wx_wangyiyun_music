# 微信小程序仿网易云音乐

#### 介绍
这是一个仿网易云的微信小程序，现在分享出来，大家一起互相学习学习^_^~~
基于其他水印工具做部分美化ui，代码内做了操作记录，对于学习者有很大帮助

对你有帮助请点个star吧~
